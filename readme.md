
# Productos Km0

An open source initiative to provide an easy way to connect farmers with customers 
without an intermediary.

## Technologies

Project is created with:

 - React Native

## Dependencies

 - Expo 
 - Expo Status Bar
 - React 
 - React DOM
 - React Native
 - React Native Web

## Installation

 - Clone repository
 - Open folder and run npm install or yarn install

## Run Project

```
cd productos/km0
npm install
npm start
```
